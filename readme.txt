== Description ==

An easy way to add testimonials to your Wordpress site.

###What you get when you use the Testimonials Box plugin

*  Place a testimonial with a simple shortcode [testimonial_box id="%postId%"]
*  Put testimonial box on any post or page.
*  Testimonial title
*  Testimonial background color
*  Testimonial description
*  Unlimited Testimonials

###Add Testimonial to any post or page

Whether you are writing a post about your company or another, you can include a well formatted testimonial.

###Quick Start Guide

* [Testimonials Box quick start guide at thinklandingpages.com] (http://www.thinklandingpages.com/testimonials-box-plugin/)

###Unlimited Testimonials

Add as many testimonials to your site as you like.  The Testimonials Box plugin lets you create testimonials and save them for when ever you are ready to use them.



== Installation ==


1. Upload `testimonials-box` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click **Testimonials Box** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The plugin does the programming and design.

= Is there a limit to the number of testimonials I can create? =
You can create an unlimited number of testimonials.

= Can the testimonials be put on my homepage? =
Yes, you can put testimonials on your homepage by using the testimonials box plugin shortcode on you homepage.

= Can I put testimonials on any post or page? =
Yes, use the testimonials shortcode, [testimonials_box id="%postId%"] on your pages and posts.
